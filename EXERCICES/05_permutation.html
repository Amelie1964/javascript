Permutation de deux variables

Ecrire un programme permutation.js où on fait permuter la valeur de deux variables, avec comme conditions initiales:

var nombre1 = 5;

var nombre2 = 3;

Après permutation, le résultat affiché doit être:

var nombre1 = 3

var nombre2 = 5

Visualiser sur la console Web l'effectivité de la permutation par la fonction console.log()

Ecrire le résultat de la permutation dans le fichier html avec la fonction document.write()

Répéter ce script avec cette fois des valeurs différentes que l'on invite l'usager à saisir dans une boite de dialogue de saisie de données
